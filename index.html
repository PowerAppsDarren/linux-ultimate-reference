Terminal Shortcuts

Ctrl+E Move cursor to end of line

Advanced Tips and Tricks

Command Chaining

Operator Description Example
cmd1 ; cmd2 Run cmd1 then cmd2 echo "Hello" ; echo "World"
cmd1 && cmd2 Run cmd2 only if cmd1 succeeds mkdir dir && cd dir
cmd1 || cmd2 Run cmd2 only if cmd1 fails ping -c1 google.com || echo "Offline"
cmd1 | cmd2 Pipe output of cmd1 to cmd2 ls -l | grep ".txt"
cmd & Run cmd in background firefox &
(cmd1 ; cmd2) Group commands (cd dir && ls -l) | grep ".txt"

Redirections

Operator Description Example
cmd > file Redirect stdout to file (overwrite) ls > files.txt
cmd >> file Redirect stdout to file (append) echo "text" >> log.txt
cmd 2> file Redirect stderr to file find / -name "file" 2> errors.txt
cmd &> file Redirect both stdout and stderr find / -name "file" &> output.txt
cmd < file Use file as stdin sort < unsorted.txt

Useful Aliases

Add these to your ~/.bashrc file:

# Navigation shortcuts
alias ll='ls -la'
alias la='ls -A'
alias l='ls -CF'
alias ..='cd ..'
alias ...='cd ../..'
alias ....='cd ../../..'

# System shortcuts
alias update='sudo apt update && sudo apt upgrade'
alias install='sudo apt install'
alias remove='sudo apt remove'
alias sysinfo='uname -a && lsb_release -a'
alias diskspace='df -h'
alias meminfo='free -m'
alias ports='netstat -tulanp'

# Safety nets
alias rm='rm -i'
alias cp='cp -i'
alias mv='mv -i'

Cron Jobs

Edit crontab with:

crontab -e

Cron job format:

minute hour day_of_month month day_of_week command

Examples:

# Run daily at 3:30 AM
30 3 * * * /path/to/script.sh

# Run every Monday at 9:00 AM
0 9 * * 1 /path/to/script.sh

# Run every 15 minutes
*/15 * * * * /path/to/script.sh
