Ubuntu Linux Commands Cheat Sheet

Getting Started

Terminal Shortcuts

Shortcut Description
Ctrl+Alt+T Open terminal
Ctrl+C Interrupt/kill current process
Ctrl+Z Suspend current process
Ctrl+D Log out of current session
Ctrl+R Search command history
Ctrl+L or clear Clear terminal screen
Ctrl+A Move cursor to beginning of line
Ctrl+E Move cursor to end of line
Ctrl+U Cut text from cursor to start of line
Ctrl+K Cut text from cursor to end of line
Ctrl+W Cut the word before the cursor
Ctrl+Y Paste text that was cut
Tab Auto-complete commands or filenames
Up/Down Arrow Navigate through command history

Basic Navigation

Command Description Example
pwd Print working directory pwd
cd directory Change to specified directory cd Documents
cd Change to home directory cd
cd .. Move up one directory cd ..
cd - Return to previous directory cd -

Help Commands

Command Description Example
man command Display manual for command man ls
command --help Display help for command ls --help
info command Display command documentation info grep
whatis command Display one-line command description whatis ls

File System Navigation

Listing Files and Directories

Command Description Example
ls List directory contents ls
ls -l List with detailed information ls -l
ls -a List all files (including hidden) ls -a
ls -la Combine -l and -a options ls -la
ls -lh List with human-readable file sizes ls -lh
ls -R List recursively (include subdirectories) ls -R
ls -t Sort by modification time ls -t

Finding Files and Directories

Command Description Example
find dir -name pattern Find files by name find /home -name "*.txt"
find dir -type d -name pattern Find directories by name find /home -type d -name "Documents"
locate pattern Find files using database locate "*.conf"
which command Show full path of command which python3
whereis command Locate binary, source, and manual whereis grep

File Management

Creating Files and Directories

Command Description Example
touch file Create empty file or update timestamp touch newfile.txt
mkdir dir Create directory mkdir newdir
mkdir -p dir1/dir2/dir3 Create nested directories mkdir -p path/to/dir

Copying, Moving, and Renaming

Command Description Example
cp source dest Copy file cp file.txt backup.txt
cp -r source dest Copy directory recursively cp -r dir1 dir2
mv source dest Move or rename file/directory mv file.txt newname.txt
mv source dest/ Move file to directory mv file.txt Documents/

Deleting Files and Directories

Command Description Example
rm file Remove file rm file.txt
rm -i file Remove with confirmation rm -i file.txt
rm -r dir Remove directory recursively rm -r dir
rm -rf dir Force remove directory recursively rm -rf dir
rmdir dir Remove empty directory rmdir emptydir

File Permissions

Command Description Example
chmod permissions file Change file permissions chmod 755 script.sh
chmod u+x file Add execute permission for user chmod u+x script.sh
chown user:group file Change file owner and group chown john:users file.txt
chgrp group file Change file group chgrp users file.txt

Permission Numbers

Number Permission Description
0 --- No permissions
1 --x Execute only
2 -w- Write only
3 -wx Write and execute
4 r-- Read only
5 r-x Read and execute
6 rw- Read and write
7 rwx Read, write, and execute

Text Manipulation

Viewing File Contents

Command Description Example
cat file Display file contents cat file.txt
less file View file with pagination less large_file.txt
more file View file page by page more large_file.txt
head file Display first 10 lines head file.txt
head -n N file Display first N lines head -n 20 file.txt
tail file Display last 10 lines tail file.txt
tail -n N file Display last N lines tail -n 20 file.txt
