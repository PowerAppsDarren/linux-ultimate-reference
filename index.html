Getting Started

Terminal Shortcuts

Shortcut Description
Ctrl+Alt+T Open terminal
Ctrl+C Interrupt/kill current process
Ctrl+Z Suspend current process
Ctrl+D Log out of current session
Ctrl+R Search command history
Ctrl+L or clear Clear terminal screen
Ctrl+A Move cursor to beginning of line
Ctrl+E Move cursor to end of line
Ctrl+U Cut text from cursor to start of line
Ctrl+K Cut text from cursor to end of line
Ctrl+W Cut the word before the cursor
Ctrl+Y Paste text that was cut
Tab Auto-complete commands or filenames
Up/Down Arrow Navigate through command history

Git Commands

Getting Started with Git

Command Description Example

git status Show the working tree status git status
git add file Add a file to the staging area git add README.md
git commit -m "message" Commit changes with a message git commit -m "Initial commit"

git pull Fetch and merge changes from the remote repository git pull

git checkout branch Switch to a different branch git checkout feature-branch
git merge branch Merge a branch into the current branch git merge feature-branch
git log Show commit logs git log
git diff Show changes between commits, commit and working tree, etc. git diff

Shell Scripting Tips

Tip Description
#!/bin/bash Specify script interpreter
chmod +x script.sh Make script executable
./script.sh Run script in current shell
source script.sh Run script and apply changes to current shell
echo "text" > file Write text to file
echo "text" >> file Append text to file
read var Read user input into variable
if [ condition ]; then Start of an if statement
fi End of an if statement
for var in list; do Start of a for loop
done End of a loop
function_name() {} Define a function
return Return from a function
